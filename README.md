# 灵算批量兑换

用于批量兑换灵算兑换码。

## 安装

1. 在 Chrome 地址栏打开 `chrome://extensions`。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目根目录（包含 `manifest.json` 的文件夹）。

## 使用

1. 在浏览器中登录灵算，并打开任意 `https://lingsuan.top/` 页面。
2. 刷新该页面。
3. 确认插件图标变为蓝色后，点击浏览器工具栏中的“灵算批量兑换”图标。
4. 点击“刷新灵算页面并捕获 Authorization”。保持插件弹窗打开，状态自动显示“已捕获 Authorization”后即可继续。
5. 在输入框中粘贴兑换码，每行一个；空行会自动忽略，行首尾空白会自动去除。
6. 设置并发数（1–10），点击“开始兑换”。
7. 等待结果展示。成功与失败结果会按输入顺序显示。

## 注意事项

- 蓝色图标表示当前页面可以使用插件；灰色图标表示当前页面不可用，点击后会提示需要在 `lingsuan.top` 页面中使用。
- 必须在已登录的 `https://lingsuan.top/` 页面中使用。
- 每个兑换码只会请求一次，不会自动重试。
- 单个请求超过 30 秒未响应时，会显示为失败。
